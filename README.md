# Restaurant-Dashboard-🍽️ 

A responsive **Restaurant Dashboard** built with **HTML, CSS, and JavaScript**.  
It provides an overview of restaurant operations including reservations, orders, and revenue, with interactive charts powered by **Chart.js**.


## 🚀 Features
- Sidebar navigation for quick access to sections
- Summary cards showing:
  - Reservations
  - Orders
  - Revenue
- Interactive line chart for restaurant statistics
- Recent orders table
- Responsive design for desktop and mobile
- Easy to extend with more charts or modules


## 🛠️ Tech Stack
- **HTML5** – structure  
- **CSS3** – styling  
- **JavaScript (ES6)** – interactivity  
- **Chart.js** – data visualization
